# madrasbank

This is a simple project i did in the first year of college to implement the basic database concepts that were taught in class. 
It includes HTML, CSS, PHP, js and mysql.

Madras bank has two portals - user portal and the admin portal.

Functions of the user portal include:
-login
-transfer money
-apply for loans
-view transaction history
-view account details
-send feedback

Functions of admin portal include:
-login
-create a new user
-view loan applications and take action
-view queries and take action
